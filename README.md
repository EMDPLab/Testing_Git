# Personal GitHub Pages Site

This repository contains a simple personal website that can be hosted using [GitHub Pages](https://pages.github.com/). The main page is `index.html`, styled with `styles.css`.

## How to Use

1. Clone or fork this repository.
2. Replace the placeholder text in `index.html` with your own information.
3. Push the repository to GitHub. If you name the repo `<username>.github.io`, it will be served automatically at `https://<username>.github.io`.
4. Alternatively, enable GitHub Pages in the repository settings.

Customize the styles in `styles.css` as desired.
